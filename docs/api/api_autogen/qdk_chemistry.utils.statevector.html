Statevector utilities in qdk_chemistry.

qdk_chemistry.utils.statevector.create_statevector_from_wavefunction(wavefunction, max_dets=None, renormalize=True)[source]

Convert a wavefunction (in the form of a coefficient array) into a statevector.

Return type:

ndarray

Parameters:

  • wavefunction (Wavefunction) – The Wavefunction.

  • max_dets (int | None) – Max number of determinants to include.

  • renormalize (bool) – Whether to renormalize the statevector after filtering max number of determinants.

Returns:

A complex numpy array to represent the statevector.
