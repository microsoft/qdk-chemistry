class QuantumErrorProfile(name, description, errors=<factory>, one_qubit_gates=<factory>, two_qubit_gates=<factory>)

Bases: object

A class representing a quantum error profile containing information about quantum gates and error properties.

This class provides functionalities to define, load, and save quantum error profiles from YAML files, and convert them to Qiskit noise models for simulations.
